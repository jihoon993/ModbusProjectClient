메인 기능 (서버 <-> 클라이언트 데이터 연결 부분)

1. MainForm 로드 시 서버에서 임의로 설정된 초기 챔버 세팅값 받아오기

2. Dry button 클릭 시 서버에 Dry button On 신호 전달 후, 일정 시간 뒤 서버에서 완료 응답을 보내면 Off 처리, 클라이언트에 On/Off 표시 UI 필요

3. Control_data 설정 값 임시 저장할 List 필요

4. Control_data 창에서 설정완료 버튼을 누른 뒤, 메인화면에서 Start 버튼을 누르면 설정한 값들이 Modbus Tcp 형태로 서버에 전달됨

5. 서버에서 전달받은 Control_data 값을 정수 형태로 변환한 뒤,
   1.에서 서버에 세팅되어있던 초기 챔버 세팅값과 전달받은 값을 비교하여 일정 간격(ex 시간, 수치)으로 전달받은 값에 가까워지도록 상승시킴

6. 서버에서 값이 상승 될 때 마다 클라이언트에 데이터를 보내고, 클라이언트는 해당 값 만큼 Progress Bar(진행률 표시UI) 를 상승시킴

7. 공정 완료 시 완료 알림
